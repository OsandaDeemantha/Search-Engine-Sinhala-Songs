This is a Python web project.

To install the dependencies locally and launch it on Ubuntu, run:

sudo apt-get install python-virtualenv
virtualenv env
source env/bin/activate
pip install -r requirements.txt
python main.py

The `load-wikipedia.py` script loads an abstract.xml file into Solr, which it receives as a parameter. This file can be downloaded from http://dumps.wikimedia.org/eswiki/
